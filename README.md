# StockTest

### Features

- Search for stocks based on symbols
- See line chart data for stock prices
  - Daily
  - Weekly
  - 1 / 3 Month(s)
  - 1 / 5 Year(s)

### Cocoapods

- Firebase
- Charts

## Requirements

-   iOS 9.0+
-   Xcode 7.3
-   Cocoapods
